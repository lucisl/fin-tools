# 购物清单功能设计文档

**创建日期**: 2026-05-06  
**目标**: 为618购物节创建一个实用的购物清单管理工具

## 功能概述

创建一个实用的购物清单管理工具,支持:
- 商品分类管理(可自定义分类)
- 商品CRUD操作(增删改查)
- 购买状态跟踪(待购买/已购买/已取消)
- 预算统计(按分类、按优先级汇总)
- 导出Excel功能
- 618购物节主题优化

## 数据结构设计

### 商品数据结构

文件路径: `config/shopping_list.json`

```json
[
  {
    "id": "唯一标识(时间戳或UUID)",
    "name": "商品名称",
    "category": "分类",
    "quantity": 数量,
    "unit": "单位",
    "budget_price": 预算价格,
    "priority": "优先级(必须/想买/可选)",
    "platform": "购买平台",
    "status": "状态(待购买/已购买/已取消)",
    "actual_price": 实际价格,
    "notes": "备注",
    "created_time": "创建时间",
    "updated_time": "更新时间"
  }
]
```

### 分类配置

文件路径: `config/categories.json`

```json
{
  "categories": ["日用品", "食品", "电子产品", "服装", "美妆护肤", "其他"]
}
```

## UI界面设计

### 布局结构

**侧边栏:**
- 预算统计概览(总预算、已花费、剩余)
- 按分类筛选
- 按优先级筛选
- 按状态筛选

**主区域 - Tab标签页:**

**Tab 1: 清单管理**
- 商品卡片式展示
- 每个商品卡片显示: 商品名称、分类标签、数量、预算价格、平台、优先级标识、状态切换按钮、编辑/删除按钮
- 新增商品按钮(弹窗对话框)

**Tab 2: 统计分析**
- 总预算 vs 实际花费(进度条)
- 按分类统计(饼图或柱状图)
- 按优先级统计(柱状图)
- 购买完成率

**Tab 3: 导出功能**
- 导出当前清单为Excel
- 支持筛选后导出

### 交互设计

**新增商品:**
1. 点击"新增商品"按钮
2. 弹窗填写信息
3. 保存后刷新列表

**编辑商品:**
1. 点击商品卡片上的"编辑"按钮
2. 弹窗修改信息
3. 保存后刷新列表

**状态切换:**
- 点击状态按钮快速切换(待购买→已购买→已取消→待购买)
- 已购买商品可填写实际价格

## 核心功能模块

### 1. 商品管理模块

- `add_item()`: 新增商品
- `update_item()`: 更新商品信息
- `delete_item()`: 删除商品
- `toggle_status()`: 切换购买状态
- `get_items()`: 获取商品列表(支持筛选)

### 2. 分类管理模块

- `get_categories()`: 获取分类列表
- `add_category()`: 新增分类
- `delete_category()`: 删除分类(检查是否有关联商品)

### 3. 统计分析模块

- `calculate_budget_summary()`: 预算汇总(总预算、已花费、剩余)
- `calculate_category_stats()`: 按分类统计
- `calculate_priority_stats()`: 按优先级统计
- `calculate_completion_rate()`: 购买完成率

### 4. 数据持久化模块

- `load_data()`: 从JSON加载数据
- `save_data()`: 保存数据到JSON
- `export_to_excel()`: 导出Excel

## 技术实现

### 依赖库

- `streamlit`: Web框架
- `pandas`: 数据处理
- `openpyxl`: Excel导出
- Streamlit内置 `st.dialog()`: 弹窗对话框
- `st.session_state`: 状态管理

### 样式设计

- 使用emoji和颜色标识优先级和状态
  - 优先级: 🔴必须 / 🟡想买 / ⚪可选
  - 状态: ✅已购买 / 🛒待购买 / ❌已取消
- 响应式卡片布局
- 与现有工具风格保持一致

## 核心特性

- **数据安全**: JSON文件存储,支持备份
- **操作友好**: 弹窗式编辑,避免页面跳转
- **可视化**: 统计图表直观展示预算使用情况
- **导出灵活**: 支持筛选后导出Excel
- **618主题**: 默认分类和提示语针对购物节优化
